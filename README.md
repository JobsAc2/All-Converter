# All Converter

画像・動画・音声ファイルを別形式へ変換できるシンプルなコンバーターです。

ドラッグ＆ドロップでファイルを追加し、変換先の拡張子を選択して一括変換できます。

## 主な機能

### 画像変換

- JPG / JPEG
- PNG
- GIF
- BMP
- TIFF
- WebP
- ICO

### 動画変換

- MP4
- AVI
- MOV
- MKV
- WMV
- WebM
- MPEG / MPG
- 3GP
- TS

### 音声変換

- MP3
- WAV
- FLAC
- AAC
- OGG
- WMA
- M4A
- OPUS
- AIFF
- AU

### その他

- ドラッグ＆ドロップ対応
- 複数ファイル一括変換
- ダークテーマUI
- 変換状態表示
- 同名ファイル自動回避

---

## 動作環境

- Windows 10 / 11
- Python版または配布版EXE

### 動画・音声変換について

動画・音声変換には ffmpeg が必要です。

以下のいずれかの方法で利用できます。

- ffmpeg を環境変数(PATH)に追加する
- ffmpeg.exe をアプリと同じフォルダに配置する

---

## 使い方

1. ファイルをドラッグ＆ドロップする
2. 変換後の拡張子を選択する
3. 「変換」ボタンを押す
4. 変換完了後、元ファイルがあるフォルダを確認する
